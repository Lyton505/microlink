# microlink

## About

MicroLink is a full-stack link shortening web service using (lyton's personal domain)[https://lyton.dev]. It is built with MERN(i.e MongoDB, ExpressJS, ReactJS, NodeJS). It's a labour of love--and fun.

## Features

### Basic Features

- Link shortening
- Customizable link
- QR code generation

### Additional Features

- Link expiration
- Link password protection
- Text file attachment(max 500kb)
- Link analytics
- User authentication

## Tech Stack & Tools

- Database:
  - MongoDB Atlas
- Frontend:
  - ReactJS
- Backend:
  - ExpressJS
  - NodeJS
- Testing:
  - Jest
- Email Delivery:
  - Sendgrid
- Storage:
  - AWS S3 Bucket
